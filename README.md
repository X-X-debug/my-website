# Mood-to-Art Generator 🎨

This is a creative website that turns your mood into abstract art.  
It uses **HTML, CSS, and JavaScript** to generate colorful patterns based on how you feel.

## ✨ Features
- Simple and lightweight (no frameworks).
- Responsive design for mobile and desktop.
- Mood-based art generation.
- Option to download and share artwork.
